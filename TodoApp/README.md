
# TodoList Console Application

Полнофункциональное консольное приложение для управления списком задач с поддержкой многопользовательского режима, отката/повтора операций, шифрования данных, синхронизации с сервером и хранением в SQLite.

## Функциональность

### Основные возможности

- **Управление профилями пользователей**
  - Создание новых профилей с логином и паролем
  - Вход в существующий профиль
  - Выход из профиля

- **Управление задачами**
  - Добавление задач (однострочный и многострочный режимы)
  - Просмотр списка задач с различными параметрами
  - Чтение полного текста задачи
  - Обновление текста задачи
  - Удаление задачи
  - Изменение статуса задачи

- **Статусы задач**
  - NotStarted (не начато)
  - InProgress (в процессе)
  - Completed (выполнено)
  - Postponed (отложено)
  - Failed (провалено)

- **Расширенные функции**
  - Отмена последней операции (Undo)
  - Повтор отменённой операции (Redo)
  - Демонстрация LINQ операций
  - Демонстрация обработки ошибок
  - Демонстрация асинхронности и многопоточности
  - Параллельные загрузки с прогресс-барами
  - Синхронизация с сервером (push/pull)

### Хранение данных

- **SQLite + Entity Framework Core** — основное хранилище
- Связь между профилем и задачами — один-ко-многим
- Все данные сохраняются в файл `todos.db`

## Команды

```
help                           - показать справку
profile [-o]                   - показать профиль / выйти из профиля
add "текст"                    - добавить задачу
add -m/--multiline             - добавить задачу в многострочном режиме
view [-i] [-s] [-d] [-a]       - показать задачи
  -i  --index                  - показать индекс
  -s  --status                 - показать статус
  -d  --update-date            - показать дату изменения
  -a  --all                    - показать всё
read <idx>                     - показать полный текст задачи
status <idx> <статус>          - изменить статус задачи
update <idx> "новый текст"     - обновить текст задачи
delete <idx>                   - удалить задачу
undo                           - отменить последнее действие
redo                           - повторить отменённое действие
linq                           - демонстрация LINQ операций
error                          - демонстрация обработки ошибок
load <кол-во> <размер>         - имитация параллельных загрузок
sync --pull                    - загрузить данные с сервера
sync --push                    - отправить данные на сервер
exit                           - выход из программы
```

## Примеры использования

### Добавление задачи

```
> add "Купить молоко"
✓ Задача добавлена: Купить молоко

> add -m
Введите строки задачи (завершите вводом '!end'):
> Первая часть
> Вторая часть
> !end
✓ Задача добавлена: Первая часть\nВторая часть
```

### Просмотр задач

```
> view
Список всех задач (только текст)

> view -i -s
Список с индексами и статусами

> view -a
Список со всеми параметрами
```

### Изменение статуса

```
> status 0 inprogress
✓ Статус задачи изменён на: InProgress

> status 0 completed
✓ Статус задачи изменён на: Completed
```

### Отмена и повтор

```
> delete 1
✓ Задача удалена

> undo
↶ Отменено удаление задачи

> redo
✓ Задача удалена
```

### Демонстрация LINQ

```
> linq
Показывает: Where, OrderBy, GroupBy, Select, отложенное выполнение
```

### Параллельные загрузки

```
> load 3 100
Загрузка 1: [########--------] 40%
Загрузка 2: [######----------] 30%
Загрузка 3: [############----] 60%

✅ Все загрузки завершены.
```

### Синхронизация с сервером

```
# Отправка данных на сервер
> sync --push

# Загрузка данных с сервера
> sync --pull
```

## Хранение данных

### База данных SQLite

Файл: `todos.db`

**Таблица Profiles:**
| Поле | Тип | Описание |
|------|-----|----------|
| Id | GUID | Первичный ключ |
| Login | string | Логин (уникальный) |
| Password | string | Пароль |
| FirstName | string | Имя |
| LastName | string | Фамилия |
| BirthYear | int | Год рождения |

**Таблица Todos:**
| Поле | Тип | Описание |
|------|-----|----------|
| Id | int | Первичный ключ |
| Text | string | Текст задачи |
| Status | enum | Статус задачи |
| LastUpdate | DateTime | Дата последнего изменения |
| ProfileId | GUID | Внешний ключ к Profile |

### Связь между таблицами

Один профиль может иметь много задач (связь один-ко-многим).

## Установка и запуск

### Требования
- .NET 8.0 SDK
- SQLite (встроенная)

### Команды

```bash
# Клонирование репозитория
git clone https://github.com/dentvin/Serbin-Danil-3832.git
cd Serbin-Danil-3832/todoapp

# Установка пакетов
dotnet restore

# Создание базы данных
dotnet ef migrations add InitialCreate
dotnet ef database update

# Запуск
dotnet run
```

### Запуск сервера для синхронизации

```bash
cd ../TodoApp.Server
dotnet run
```
